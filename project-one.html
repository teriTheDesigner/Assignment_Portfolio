13C.01.01 Fonts and UI components

In this assignment I worked with creating a font hierarchy and designing UI cards.

Type Scale

Font hierarchy

UI Components

The solution

For this project, I started by selecting a sans serif font from Google Fonts that would work well for both body text and headlines. Then I used a type scale to determine the appropriate font size for different sections, paragraphs, and headlines. I also decided on a button style to use throughout the interface. To layout the UI cards, I used CSS flex in the direction of column, and the direction of row for the buttons. Finally, I implemented a font hierarchy within the UI cards to emphasize important information and create a more visually pleasing design. I decided to use a white background color for the cards to create a contrast with the gray background of the section. This way, it is easy to see the specific cards without the need for a border or a shadow for each card.
